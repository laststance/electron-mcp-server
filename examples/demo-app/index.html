üí° MCP Server Usage

Getting Started:

  1. Make sure this app is running with npm run dev
  2. Verify DevTools Protocol is enabled on port 9222
  3. Use MCP tools to interact with this app

Example MCP Commands:

// Get page structure
{
  "command": "get_page_structure"
}

// Click button by text
{
  "command": "click_by_text",
  "args": { "text": "Create New Item" }
}

// Fill input by placeholder
{
  "command": "fill_input",
  "args": {
    "placeholder": "Enter your name",
    "value": "John Doe"
  }
}

// Select country
{
  "command": "click_by_selector",
  "args": { "selector": "#country" }
}
